Output

string Output([string name [, string dest]])

版本

1.0

內容

送出文件到指定目標，例如：字串，本地檔案或瀏覽器。 在最後一個步驟，可能會用到外掛程式("若存在")或強行則行下載("另存" 對話盒)。
若需要終斷文件，Output()這個功能會首先則行Close()。

參數

name
檔案名稱。如果沒有提供檔案名稱，當文件存送到瀏覽器(dest I)時，檔案名稱會自動定義為doc.pdf。
dest
送出文件的目標位置。可用下面任何一個值：
  • I：送出檔案到瀏覽器上。如果有外掛程式存在的話，也可同時使用。使用者可透過連結去另存PDF檔案，檔案名稱會由 (name 參數)定義。
  • D：存送到瀏覽器，並且強行下載存檔，檔案名稱會由 (name 參數)定義。
  • F：儲存到本機檔案，檔案名稱會由 (name 參數)定義。
  • S：透過字串反回文件，檔案名稱不需要使用。假如，dest的參數沒有定義，但已經定義了檔案名稱，那 dest的參數會自動定義為F。若果什麼參數都沒有定義的話，那 dest的參數會自動定義為I。

    注意：為了相容之前的版本，boolean 值是也會接受的。(錯誤為F、正確為D)
