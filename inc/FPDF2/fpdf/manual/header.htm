Header

Header()

版本

1.0

內容

這個功能是用來產生頁首。AddPage() 和 Close() 會自動呼叫這個功能，但程式不會直接呼叫這個功能。這個結果在 FPDF 裡是空白的，若果你想進行特殊處理，你要透過(subclass)去撤銷這個功能。

例子

class PDF extends FPDF
{
function Header()
{
    //Select Arial bold 15
    $this->SetFont('Arial','B',15);
    //Move to the right
    $this->Cell(80);
    //Framed title
    $this->Cell(30,10,'Title',1,0,'C');
    //Line break
    $this->Ln(20);
}
}
