SetDisplayMode

SetDisplayMode(mixed zoom [, string layout])

版本

1.2

內容

定義文件在顯示出來的模式。可以設定(縮小放大)的級數：使到顯示出來的頁面為全畫面、視窗的全寬度、使用真正的大小、安比例(縮小放大)或使用預設值(已Acrobat作為標準定義)。 還可以具體規劃頁面：只有一頁、連續顯示、兩個欄位或瀏覽程式的預設值。 預設值：文件使用全寬度和連續顯示模式。

參數

zoom
使用(縮小放大)功能。這個功能可以使用已下的數值：
  • fullpage: 使到顯示出來的頁面為全畫面
  • fullwidth: 視窗的全寬度(用視窗的最大寬度)
  • real: 使用真正的大小 (相等於 100% (縮小放大))
  • default: 使用瀏覽程式的預設值
或用數值來定義(縮小放大)參數
layout
頁面規劃。這個功能可以使用已下的數值：
  • single: 只有一頁
  • continuous: 連續顯示
  • two: 顯示兩頁在兩個欄位上
  • default: 使用瀏覽程式的預設值
預設值：連續顯示
