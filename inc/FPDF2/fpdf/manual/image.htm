Image

Image(string file, float x, float y [, float w [, float h [, string type [, mixed link]]]])

版本

1.1

內容

把一個圖像放在頁裡面。圖像的左上角作為定點座標。尺寸定義可以使用不同的方法：
  • 清楚的寬度和長度(自行定義尺寸單位)。
  • 詳盡的尺寸：為了保持圖像基本尺寸比例，它會自動計算尺寸，
  • 是沒有需要定義詳盡尺寸。在這個事例，圖象定為每寸72點(72 dpi)。
支援 JPEG 和 PNG 格式。

JPEG，所有支援模式：
  • gray scales
  • true colors (24 bits)
  • CMYK (32 bits)
PNG，所有支援模式：
  • gray scales 最大支援 8 bits (256 levels)
  • indexed colors
  • true colors (24 bits)
但不支援：
  • Interlacing (交錯式)
  • Alpha channel
它會很重視透明色的定義。(但只限於 Acrobat 4 或已上版本). 圖像格式可以自行定義或由圖像檔案名稱後面的副檔名來推論出圖像的格式。 它可以加入連結在圖像上。 備註：若果一個圖像會用上幾次，只需要一個圖像檔案便足夠。

參數

file
圖像檔案名稱
x
左上角的橫座標(x)
y
左上角的縱座標(y)
w
在頁面上圖像的寬度。如果沒有定義或定義為0，它會自動計算。
h
在頁面上圖像的高度。如果沒有定義或定義為0，它會自動計算。
type
圖像格式。格式值為：JPG、JPEG、PNG。如果沒有定義圖像格式，會由圖像檔案名稱後面的副檔名來推論出圖像的格式。
link
URL 或 identifier 由 AddLink() 處理。
